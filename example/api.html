    ChCalendar基类

    配置项

    date

    当前默认选中日期设置；

    类型：Object
     属性：

    • start 类型：Date；默认值：当前时间；开始日期
    • end 类型：Date；默认值：当前时间；结束日期

    如是单选日历则只需要设置start值即可；

    currentDate

    当前日历面板显示哪个日期的日期表格

    类型：Date
    默认值：当前起始日期
     可传任何有效的时间格式，如（new Date() || "2017/01/01" || 毫秒数）；

    intervals

    时段选择相关配置项；

    类型：Object
     属性：

    • show 类型：Boolean；默认：true；是否开启时段选择功能；
    • max 类型：Number；默认：30；最多能选多少天；
    • min 类型：Number；默认：1；最少需要选多少天，值为1-n；
    • first 类型：Date； 默认：空；起始可用时间，在这时间之前的日期将为禁用状态。可传任何有效的时间格式；
    • last 类型：Date； 默认：空；可用时间结束，在这时间之后的日期将为禁用状态。可传任何有效的时间格式；

    disable

    指定需要禁用的日期，经常用在如某日预约已满，需要禁用这日预约的时候。

    类型：Array
     默认值：无
     示例：["2015/05/02","2015/10/01","2015/10/02"]

    format

    类型：String
     默认值：yyyy/mm/dd
     时间相应字符：y（年） m（月） d（日）h（时） i（分） s（秒）

    type

    日历类型，支持日历和月历

    类型：String
     默认值：day
     支持类型：day || month

    disableSelect

    日期是否可以点选开关，如只是做日期展示，可把此开关关闭；

    类型 ： Boolean
     默认值：true

    supplement

    日历上下月日期补齐。是否在当前月日历表格的空白格子里，补齐邻月日期；

    类型：Boolean
     默认：true

    preNextButton

    上下（月||年）按钮开关；

    类型：Object
     属性：

    • pre 类型：Boolean；默认：true；上按钮；
    • next 类型：Boolean；默认：true；下按钮；

    classPrefix

    class样式前缀;

    类型：String
     默认：Ch

    language

    类型：String
     默认值：zh_CN
     支持类型：zh_CN || en_US
     日历显示语言

    实例方法

    setOptions

    当前实例方法，可以在任何时候调用此方法重置当前实例所有配置项中的属性。除改变显示面版类型，需要改变面板显示类型请使用setCurrentPanels方法

    参数：（options,callback）

    • options[Object]格式参照 配置项
    • callback[function(opt)] 设置完成的回调方法。回传参数opt[Object]属性只有msg，值包括 success（成功）disable（设置的时间有禁用的时间）error（其他错误）

    用法： 

    <div id='my_date'></div>
<div id='set_date'>设置选中时间为：2017/01/01-2017/01/10</div>
var myDate = new ChCalendar('#my_date',{}); //创建日历
$('#set_date').click(function(){
    myDate.setDates({date : {
        start : '2017/01/01',
        end : '2017/01/10'
        }},function(opt){
            console.log(opt.msg) //回传信息  success成功、disable设置的时间包含禁用的时间、error其他错误
            })
})

    setDates

    手动重置当前日历选中日期。setDates为当前实例方法，如需要在其他地方联动当前日历时间，可调用此方法。

    参数：（options,callback）

    • options[{date}]格式参照 date
    • callback[function(opt)] 设置完成的回调方法。回传参数opt[Object]属性只有msg，值包括 success（成功）disable（设置的时间有禁用的时间）error（其他错误）

    用法： 

    <div id='my_date'></div>
<div id='set_date'>设置选中时间为：2017/01/01-2017/01/10</div>
 var myDate = new ChCalendar('#my_date',{}); //创建日历
 $('#set_date').click(function(){
     myDate.setDates({
         start : '2017/01/01',
         end : '2017/01/10'
         },function(opt){
             console.log(opt.msg) //回传信息  success成功、disable设置的时间包含禁用的时间、error其他错误
             })
})

    setCurrentPanels

    设置当前显示面板信息，setCurrentPanels为当前实例方法，如需要在其他地方联动当前日历面板显示，可调用此方法。

    参数：（options）

    • options[{currentDate,type}] 请参照renderType type

    用法： 

    <div id='my_date'></div>
<div id='set_day'>显示2017年1月日期</div>
<div id='set_month'>显示2017年月历</div>
 var myDate = new ChCalendar('#my_date',{}); //创建日历
 $('#set_day').click(function(){
     myDate.setCurrentPanels({//调用setCurrentPanels方法显示到2017年01月
         currentDate : '2017/01/01',
         type : 'day'
         })
})
 $('#set_month').click(function(){
     myDate.setCurrentPanels({//调用setCurrentPanels方法显示2017年月历
         currentDate : '2017',
         type : 'month'
         })
})

    实例事件

    所有事件都通过jquery的on方法监听

    changeDay

    面板显示日历，选择日期后的事件

    参数：String [date] 当前点击的日期
     用法： 

    var myDate = new ChCalendar('target',{});
$(myDate).on("changeDay",function(e,data){
    console.log(data) // 2017/01/01
})

    changeMonth

    面板显示月历，选择某月的事件

    参数：String [date] 当前点击的日期
     用法： 

    var myDate = new ChCalendar('target',{});
$(myDate).on("changeMonth",function(e,data){
    console.log(data) // 2017/01
})

    changeYear

    面板显示年历，选择某年后的事件

    参数：String [date] 当前点击的日期
     用法： 

    var myDate = new ChCalendar('target',{});
$(myDate).on("changeYear",function(e,data){
    console.log(data) // 2017
})

    pre

    点击上（月｜年）按钮后的事件

    参数：Object [{type,currentDate}] type：当前显示面板显示类型；currentDate：当前面板显示的日期
     用法： 

    var myDate = new ChCalendar('target',{});
$(myDate).on("pre",function(e,data){
    console.log(data.type,data.currentDate) //   day , 2017/01/01
})

    next

    点击下（月｜年）按钮后的事件

    参数：Object [{type,currentDate}] type：当前显示面板显示类型；currentDate：当前面板显示的日期
     用法： 

    var myDate = new ChCalendar('target',{});
$(myDate).on("next",function(e,data){
    console.log(data.type,data.currentDate) //   day , 2017/01/01
})

    util方法库

    util类是ChCalendar构造函数的静态方法类，使用：ChCalendar.util.方法名

    formatDate

    时间格式化

    参数：

    • date
    • format y年m月d日 h时i分s秒

    return：String
     用法： 

    ChCalendar.util.formatDate(new Date(),"yyyy-mm-dd hh:ii:ss") //2017-01/01 00:00:00

    leapPingnian

    判断平年闰年

    参数：year 当前年
     return：Boolean ［true 闰年 ／false 平年］

    monthHasDay

    计算本月有多少天

    参数：

    • year 年
    • month 月

    return：Number

    isWeeks

    计算日期是周几

    参数：

    • date 日期，默认为今日

    return：Number （1-7）

    addZero

    个位数补0

    参数：

    • number 数字（1-9）

    return：String（01-09）

    calculationTime

    计算日期加减日

    参数：

    • date 被计算日期
    • number 需要加减多少日 +1 -1（正负区分加，减）

    return : Date

    calculationMonth

    计算加减n月

    参数

    • date 被计算日期
    • number 需要加减多少月 +1 -1（正负区分加，减）

    return : Date

    testDate

    检测B时间数组是否包含A时间

    参数：

    • date A时间
    • array B时间数组

    return：Boolean

    testMacthMax

    判断A时间是否大于B时间

    参数：

    • dateA
    • dateB

    return：Boolean

    testMacthMin

    判断A时间是否小于B时间

    参数：

    • dateA
    • dateB

    return：Boolean

    getDateInterval

    获取A时间和B时间中间的日期List

    参数：

    • dateA
    • dateB

    return：Array
