    基础

    介绍

    ChCalendar是一个基于jQuery开发的日历控件。他的核心只有一个ChCalendar构造函数，他的实例是一个基础日历面板，详细请看基础日历教程。其他所有日历选择器（单日历，双日历）都是ChCalendar构造函数的扩展。如果插件已有的扩展不满足您的需求，您也可以基于ChCalendar扩展出自己想要的日历选择器。

    使用

    ChCalendar是基于jQuery开发，使用前确保已引入jQuery库。
     直接下载并用 script 标签引入，ChCalendar 会被注册为一个全局构造函数。
     支持AMD CMD引入，对外输出的也是ChCalendar构造函数。

    下载地址

    https://github.com/cwlch/ChCalendar

    一次性引入

    文件目录：dist/ChCalendar[version].js
     示例： 

    <script src='./dist/ChCalendar1.0.1.js'>
<input id='normal'/>
<input id='double'/>
<div id='core'></div>
<script>
new ChCalendar('#core');//创建基础日历
new ChCalendar.normal('#normal');//单日历选择器
new ChCalendar.double('#double');//双日历选择器
</script>

    模块加载

    核心文件：dist/ChCalendar[version]-base.js
     模块文件：dist/module/*.js
     示例： 

    <script src='./dist/ChCalendar1.0.1-base.js'>
<script src='./dist/modle/Normal.js'>
<script src='./dist/modle/Double.js'>
<input id='normal'/>
<input id='double'/>
<script>
new ChCalendar.normal('#normal');//单日历
new ChCalendar.double('#double');//双日历
</script>

    扩展

    单日历选择器

    创建单日历

    var myDate = new ChCalendar.normal('#normal');//单日历

    双日历选择器

    创建双日历

    var myDate = new ChCalendar.double('#normal');//单日历

    基础日历

    创建基础日历

    基础日历只需要引入ChCalendar[version]-base.js文件就可以支持。
     直接new ChCalendar的实例就是一个基础日历。

    ChCalendar函数共接受俩个参数。第一个DOM容器，第二个配置项
     DOM容器：因为基础日历并不是一个选择器，所以我们的DOM节点不是input框。而是一个拿来放日历表格的DOM容器。
     配置项：详细请查看APIChCalendar基类->配置项 

    <script src='ChCalendar1.0.1-base.js'></script>
<div id='base'></div>//日历表格容器
var myDate = new ChCalendar('#base',{
    date:{
        start:'2017-01-01', //初始选中起始时间
        end:'2017-01-05'//初始选中结束时间
        },
        format:'yyy-mm-dd',//日期格式
    });

    获取日期

    配置项中所有信息都可以通过实例的o属性获取当前实例的实时信息。 

    var myDate = new ChCalendar('#base',{
    date : {start : '2017/01/01',end:'2017/01/05'}
});
myDate.o.date //{start : '2017/01/01',end:'2017/01/05'}//获取实时日期，如中间有点选日期，将会返回选中后的日期

    重置配置项&setOptions

    setOptions方法可以重置所有配置项中的配置，方便各种额外业务操作日历。

    重置日期

    我们经常会使用一些快速选择按钮，或者日历需要关联其他地方日期。 

    <div id='btn'>选择2017-10-01－2017-10-05<div>
<div id='btn2'>昨日<div>
var myDate = new ChCalendar('#base');
$("#btn").click(function(){
    myDate.setOptions({//快速选择其他时间
        date:{
            start : '2017-10-01',
            end : '2017-10-05'
        }
    }
});
var toDay = ChCalendar.util.calculationTime(new Date(),-1)//计算时间我们可以使用util方法库中的方法
$("#btn2").click(function(){
    myDate.setOptions({//快速选择昨日
        date:{
            start : toDay
            end : toDay
        }
    }
})

    重置显示语言

    <div id='btn'>英文<div>
<div id='btn2'>中文<div>
var myDate = new ChCalendar('#base');
$("#btn").click(function(){
    myDate.setOptions({
        language:'en_US'
    }
})
$("#btn2").click(function(){
    myDate.setOptions({
        language:'zh_CN'
    }
})

    重置日期格式

    <div id='btn'>yyyy年mm月dd日<div>
<div id='btn2'>yyyy／mm／dd<div>
var myDate = new ChCalendar('#base');
$("#btn").click(function(){
    myDate.setOptions({
        format:'yyyy年mm月dd日'
    }
})
$("#btn2").click(function(){
    myDate.setOptions({
        format:'yyyy／mm／dd'
    }
})
